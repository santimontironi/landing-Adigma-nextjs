# Documentación Landing Page - Adigma (en desarrollo)

## Descripción General

Landing page moderna y profesional para **Adigma**, empresa especializada en implementación de Odoo. La página está construida con **Next.js** y **Tailwind CSS**, ofreciendo una experiencia fluida y responsive.

### Tecnologías Utilizadas

- **Framework:** Next.js 16 (React)
- **Estilos:** Tailwind CSS
- **Iconos:** Bootstrap Icons
- **Imágenes:** Next/Image (optimización automática)
- **Lenguaje:** JavaScript/JSX
- **Base de datos:** PostgreSQL
